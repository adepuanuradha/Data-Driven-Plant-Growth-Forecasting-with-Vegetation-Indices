# Data-Driven-Plant-Growth-Forecasting-with-Vegetation-Indices
Plant Health Monitoring System using NDVI, RGB image features, Machine Learning, and Data Analysis.
## Overview
This project is a Plant Health Monitoring System that analyzes plant health using NDVI (Normalized Difference Vegetation Index) values and RGB image features extracted from plant images.
## Features
* Plant image processing using OpenCV
* RGB feature extraction from images
* NDVI-based health classification
* Health prediction using Machine Learning
* Exploratory Data Analysis (EDA)
* Health category visualization
* Treatment recommendations for unhealthy plants
## Machine Learning Models
* Support Vector Classifier (SVC)
* Gradient Boosting Classifier
* Random Forest Classifier
## Technologies Used
* Python
* OpenCV
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
## Health Categories
* Dead
* Unhealthy
* Moderately Healthy
* Very Healthy
